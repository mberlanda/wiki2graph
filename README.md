# Wiki2Graph
## A package to parse Wikimedia dump files and create graphs in Neo4j


### Instructions

```
# run the tests
/wiki2graph $ python setup.py test
# install the package
/wiki2graph $ pip install -e .
# run the script
/wiki2graph/wiki2graph $ python main.py


```
